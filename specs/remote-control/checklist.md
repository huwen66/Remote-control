# Checklist

## 环境与依赖
- [x] Python 3 环境可用，所需依赖库（pillow、pyautogui、pynput、numpy）已安装

## 公共模块 common.py
- [x] send_msg / recv_msg 能正确处理粘包（4字节长度头协议）
- [x] 鼠标/键盘事件可正确序列化与反序列化（JSON 格式）
- [x] 消息类型常量（AUTH、FRAME、EVENT）已定义

## 服务端 server.py
- [x] 服务端启动后在指定端口正常监听 TCP 连接
- [x] 密码认证：正确密码允许连接，错误密码拒绝连接并给出提示
- [x] 屏幕捕获线程持续运行，帧率不低于 5 FPS
- [x] 屏幕帧以 JPEG 格式压缩后发送，画质可接受
- [x] 接收客户端鼠标事件后，服务端正确执行鼠标移动/点击/滚轮操作
- [x] 接收客户端键盘事件后，服务端正确执行键盘输入
- [x] 服务端 GUI 正确显示本机 IP、端口、密码、当前连接状态

## 客户端 client.py
- [x] 连接界面可输入 IP、端口、密码，点击连接按钮后发起连接
- [x] 密码正确时连接成功，密码错误时弹出错误提示
- [x] 连接成功后客户端窗口实时渲染服务端桌面画面
- [x] 客户端鼠标移动事件正确映射坐标并发送
- [x] 客户端鼠标点击（左键、右键、双击）事件正确发送
- [x] 客户端鼠标滚轮事件正确发送
- [x] 客户端键盘按键事件正确发送

## 集成演示
- [x] 在同一台机器上可成功运行 server.py 和 client.py 演示远程控制全流程
- [x] 断开连接后服务端恢复等待连接状态，客户端界面恢复连接输入界面
